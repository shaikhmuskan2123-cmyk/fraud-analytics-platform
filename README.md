# Fraud Analytics Platform

## Overview

Fraud Analytics Platform is a machine learning-based fraud detection system built using Python.

The platform analyzes transaction details and predicts whether a transaction is fraudulent using a trained Random Forest model.

## Features

* Fraud Detection
* Risk Scoring
* Risk Classification
* SQLite Database Storage
* Flask REST API
* Streamlit Dashboard
* PDF Report Generation
* Transaction History Tracking
* Interactive Analytics Charts

## Technologies Used

* Python
* Streamlit
* Flask
* SQLite
* Scikit-Learn
* Pandas
* Plotly
* ReportLab
